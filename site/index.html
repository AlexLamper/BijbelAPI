Render en Stripe setup

Volg deze stappen om `bijbelapi.com` en billing correct live te zetten.

Render + domein (Hostinger)

  1. 1. Deploy service met `render.yaml`.
  2. 2. Voeg `bijbelapi.com` en `www.bijbelapi.com` toe in Render custom domains.
  3. 3. Zet DNS records in Hostinger exact zoals Render aangeeft.
  4. 4. Forceer redirect `www` -> apex (`bijbelapi.com`).
  5. 5. Controleer SSL status op Render (Issued).

Stripe environment variables

  • - `STRIPE_SECRET_KEY`
  • - `STRIPE_WEBHOOK_SECRET`
  • - `STRIPE_PRICE_ID_PRO_MONTHLY`
  • - `STRIPE_PRICE_ID_PRO_YEARLY`
  • - `APP_BASE_URL=https://bijbelapi.com`
  • - `BILLING_ENFORCED=true`
  • - `FREE_TIER_DAILY_LIMIT=5000` (voorbeeld)
