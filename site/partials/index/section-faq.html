Veelgestelde vragen

Wat is BijbelAPI?▼
BijbelAPI is een Nederlandstalige REST API waarmee ontwikkelaars Bijbelteksten in JSON kunnen opvragen, inclusief onder andere verzen, passages, dagtekst en zoeken. Er is OpenAPI-documentatie op /docs.
Is er een gratis Bijbel API?▼
Ja. Er is een gratis tier voor testen en licht gebruik. Bij actieve billing kan een daglimiet per IP gelden; met een Pro-abonnement en API-sleutel krijg je hogere limieten via Stripe.
Welke Nederlandse vertalingen worden ondersteund?▼
De codebase ondersteunt meerdere coderingen zoals nbg1951, bb, sv en nld1939. Welke bestanden daadwerkelijk op de server staan, zie je via GET /api/versions.
Hoe vraag ik een specifiek vers op?▼
Gebruik GET /api/verse met queryparameters book, chapter en verse. Optioneel: parameter version voor een specifieke vertaling. Zie /docs voor alle velden.
Waar vind ik de technische documentatie?▼
Op het pad /docs staat Swagger UI met alle endpoints. ReDoc staat op /redoc.
Hoe werkt Pro en Stripe?▼
Pro gebruikt Stripe Checkout en het Customer Portal. Endpoints zoals /billing/checkout-session en /billing/portal-session staan in de OpenAPI-specificatie.
Mag ik de API commercieel gebruiken?▼
Dat hangt af van je abonnement en van de licenties op de onderliggende Bijbelteksten. Raadpleeg de pagina Bronnen en licenties en je eigen juridische advies.
