API Referentie

Snel aan de slag

Kopieer een van de codevoorbeelden en je eerste request werkt in minder dan een minuut.

Vers ophalen (GET)

curl -sS "https://bijbelapi.com/api/verse\
  ?book=Johannes&chapter=3&verse=16\
  &version=bb" \
  -H "Accept: application/json"

Met Pro API-sleutel

curl -sS "https://bijbelapi.com/api/search\
  ?q=licht&version=bb" \
  -H "Accept: application/json" \
  -H "x-api-key: JOUW-API-SLEUTEL"

Voorbeeld response

{
  "book": "Johannes",
  "chapter": 3,
  "verse": 16,
  "version": "bb",
  "text": "Want alzo lief heeft God de wereld gehad, dat Hij zijn eniggeboren Zoon gegeven heeft..."
}

fetch (browser / Node)

const res = await fetch(
  'https://bijbelapi.com/api/verse' +
  '?book=Johannes&chapter=3&verse=16' +
  '&version=bb',
  { headers: { Accept: 'application/json' } }
);
const data = await res.json();
console.log(data.text);

Met Pro API-sleutel

const res = await fetch(
  'https://bijbelapi.com/api/search?q=licht',
  {
    headers: {
      Accept: 'application/json',
      'x-api-key': 'JOUW-API-SLEUTEL',
    },
  }
);
const data = await res.json();

Voorbeeld response

{
  "book": "Johannes", "chapter": 3, "verse": 16,
  "version": "bb",
  "text": "Want alzo lief heeft God de wereld gehad..."
}

requests (Python 3)

import requests

r = requests.get(
    "https://bijbelapi.com/api/verse",
    params={
        "book": "Johannes",
        "chapter": 3, "verse": 16,
        "version": "bb",
    },
)
print(r.json()["text"])

Met Pro API-sleutel

import requests

r = requests.get(
    "https://bijbelapi.com/api/search",
    params={"q": "licht", "version": "bb"},
    headers={"x-api-key": "JOUW-API-SLEUTEL"},
)
results = r.json()

Voorbeeld response

{
  "book": "Johannes", "chapter": 3, "verse": 16,
  "version": "bb",
  "text": "Want alzo lief heeft God de wereld gehad..."
}
