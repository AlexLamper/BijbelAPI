FAQ

Veelgestelde vragen

Wat is BijbelAPI?
BijbelAPI is een Nederlandstalige REST API waarmee ontwikkelaars Bijbelteksten als JSON kunnen opvragen. Je kunt verzen, passages, de dagtekst, zoekresultaten en commentaar ophalen. Er is interactieve OpenAPI-documentatie op /docs.
Is er een gratis Bijbel API?
Ja. De gratis tier geeft 50 requests per dag per IP-adres, zonder API-sleutel. Dat is voldoende om te integreren en te testen. Voor productiegebruik of hogere limieten upgrade je naar Pro.
Waar is mijn API-sleutel na betaling?
Na een geslaagde betaling word je teruggestuurd naar bijbelapi.com. Bovenaan de pagina verschijnt een blauwe balk met je sleutel. Kopieer hem en bewaar hem veilig — je ziet hem maar eenmaal automatisch. Daarna kun je hem altijd opnieuw ophalen via GET /billing/status met je sleutel als x-api-key header.
Welke betaalmethoden worden geaccepteerd?
Betaling verloopt via Stripe Checkout, de veiligste betalingsinfrastructuur ter wereld. Afhankelijk van je regio zijn iDEAL, Mastercard en Visa beschikbaar. BijbelAPI slaat zelf geen betaalgegevens op.
Kan ik mijn abonnement opzeggen?
Ja, je kunt op elk moment opzeggen via het Stripe Customer Portal. Klik op "Open het Stripe Customer Portal" op de pricing-pagina en log in met het e-mailadres dat je bij Checkout hebt gebruikt. Na opzegging blijft je sleutel actief tot het einde van de betaalperiode.
Welke Nederlandse vertalingen worden ondersteund?
Momenteel ondersteunde vertalingen zijn o.a. NBG1951, HSV, BasisBijbel, Staten Vertaling (SV) en NLD1939. Welke bestanden actief zijn op de server zie je live via GET /api/versions.
Hoe vraag ik een specifiek vers op?
Gebruik GET /api/verse?book=Genesis&chapter=1&verse=1&version=nbg1951. Alle parameters worden uitgelegd in de Swagger-documentatie.
Waar vind ik de technische documentatie?
Op /docs staat Swagger UI met alle endpoints en request/response-voorbeelden. ReDoc staat op /redoc.
Mag ik de API commercieel gebruiken?
Dat hangt af van je abonnement en van de licenties op de onderliggende Bijbelteksten. Raadpleeg de pagina Bronnen en licenties en indien nodig juridisch advies.
