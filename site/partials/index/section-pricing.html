Pricing en abonnementen

Start gratis voor tests en kleine projecten. BijbelAPI Pro geeft hogere limieten, betaling via Stripe (iDEAL en kaarten afhankelijk van je Stripe-instellingen) en zelfservice via het klantportaal. Alle prijzen zijn inclusief btw zoals je die in Stripe hebt geconfigureerd. Hoe gebruik je Pro na aankoop?

Optioneel: sla je Pro-sleutel lokaal op zodat deze pagina “Pro actief” kan tonen. Klik Onthouden na het plakken, of klik buiten het veld — dan wordt de sleutel ook opgeslagen (zelfde effect).

Op gedeelde computers: gebruik Vergeet sleutel als je klaar bent. De homepage moet via de server geladen worden (http://127.0.0.1:8081 of je domein), niet als los HTML-bestand.

Hoe gebruik je Pro na aankoop?

Pro gebruik je overal met de HTTP-header x-api-key — niet als vaste query-parameter in de URL. Zie voorbeelden hieronder.

  1. Sleutel verkrijgen — Na Stripe Checkout eenmalig op deze site: blauwe balk met “kopieer sleutel” (sessie in de URL), of plak de sleutel die je hebt bewaard.
  2. “Pro actief” op deze pagina — Plak hierboven, klik Onthouden óf klik buiten het veld na plakken. Dan roept de pagina /billing/status aan om je abonnement te tonen.
  3. Je app of scripts — Voeg bij elke API-request de header toe: x-api-key: <jouw-sleutel>. Zo krijg je Pro-limieten zodra BILLING_ENFORCED=true op de server staat.
  4. Betalen opzeggen / facturen — Onderaan: Abonnement beheren → Stripe-klantportaal met het zelfde e-mailadres als bij Checkout.

Voorbeelden

curl (GET)

curl -sS -H "x-api-key: JOUW-SLEUTEL" \
  "{{CANONICAL_ORIGIN}}/api/versions"

curl (POST parse)

curl -sS -X POST "{{CANONICAL_ORIGIN}}/api/parse/reference" \
  -H "Content-Type: application/json" \
  -H "x-api-key: JOUW-SLEUTEL" \
  -d "{\"reference\":\"Johannes 3:16\",\"version\":\"nbg1951\"}"

JavaScript (fetch)

fetch("/api/versions", {
  headers: {
    "Accept": "application/json",
    "x-api-key": localStorage.getItem("bijbelapi_api_key"),
  },
});
